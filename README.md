# Emoji Factory

A simple web app that shows illustrates how fast products are made.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## How it works

Choose a preset (or `Custom`), set company/emoji/count/interval, and the conveyor spawns emojis at the calculated rate.  
Use `Pause Production` to stop/resume the flow, and use Share to copy a link with the current setup.
